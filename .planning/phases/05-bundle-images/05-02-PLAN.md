   ```javascript
   // ── Renderiza imagens de bundle ────────────────────────────────

   function renderBundleImages(images) {
     bundleImagesContainer.innerHTML = '';
     // images is an object like { "2": { src: "..." }, "3": { src: "..." } }
     var keys = Object.keys(images);
     if (keys.length === 0) return;

     keys.sort(function(a, b) { return Number(a) - Number(b); });

     keys.forEach(function(qty) {
       var imgData = images[qty];
       var label = bundleLabel(Number(qty)) || ('Bundle ' + qty);

       var row = document.createElement('div');
       row.className = 'bundle-image-item';

       var thumb = document.createElement('img');
       thumb.className = 'bundle-image-thumb';
       thumb.src = imgData.src;
       thumb.alt = label;
       thumb.onerror = function() { this.style.display = 'none'; };

       var labelEl = document.createElement('span');
       labelEl.className = 'bundle-image-label';
       labelEl.textContent = label;

       var input = document.createElement('input');
       input.type = 'url';
       input.value = imgData.src;
       input.placeholder = 'https://nova-imagem.com/produto.png';
       input.setAttribute('data-bundle-qty', qty);
       input.setAttribute('data-original-src', imgData.src);

       // Live-update thumbnail when user pastes new URL
       input.addEventListener('input', function() {
         var val = this.value.trim();
         if (val) {
           thumb.src = val;
           thumb.style.display = '';
         }
       });

       row.appendChild(thumb);
       row.appendChild(labelEl);
       row.appendChild(input);
       bundleImagesContainer.appendChild(row);
     });
   }
   ```

   Per D-09: Each row has thumbnail 60x60 + label + input.
   The `data-bundle-qty` and `data-original-src` data attributes enable the export payload builder.

4. Add `buildBundleImagesPayload()` function right after `buildCheckoutPayload()` (after line 585):
   ```javascript
   function buildBundleImagesPayload() {
     var result = {};
     var inputs = bundleImagesContainer.querySelectorAll('input[data-bundle-qty]');
     inputs.forEach(function(input) {
       var qty = input.getAttribute('data-bundle-qty');
       var originalSrc = input.getAttribute('data-original-src');
       var newSrc = input.value.trim();
       // Only include if user changed the URL
       if (newSrc && newSrc !== originalSrc) {
         result[qty] = { originalSrc: originalSrc, newSrc: newSrc };
       }
     });
     return result;
   }
   ```

## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| server response -> frontend state | bundleImages from /api/fetch — trusted server data, stored in state |
| user input -> thumbnail src | User pastes URL into input field, live-updates img.src — browser sandbox handles display |
| user input -> export payload | User-provided newSrc sent to server — validated server-side in Plan 05-01 T-05-01 |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-05-04 | Tampering | renderBundleImages img.src | accept | img.src assignment from user input. Browser sandbox prevents code execution from image URLs. onerror handler hides broken images. |
| T-05-05 | Spoofing | buildBundleImagesPayload newSrc | mitigate | Server-side URL validation (T-05-01 in Plan 05-01) rejects non-parseable URLs. Frontend uses `type="url"` input for basic browser validation. |
| T-05-06 | Information Disclosure | data-original-src attribute | accept | Original image URLs are from the public page being cloned. No private data. Attribute stored in DOM only, not persisted. |
</threat_model>
