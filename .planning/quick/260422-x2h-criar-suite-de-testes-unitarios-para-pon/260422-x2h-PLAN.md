**Step 3: Create test-parse-image-dims.js** following same pattern.

Test `parseImageDimensions(buffer)` by crafting minimal valid binary headers:

Test 1: PNG (100x200)
- Create Buffer with PNG signature (89 50 4E 47 0D 0A 1A 0A) + IHDR chunk.
  Bytes 16-19: width 100 as UInt32BE, bytes 20-23: height 200 as UInt32BE. Pad to 30+ bytes.
- Assert: `{ width: 100, height: 200 }`

Test 2: JPEG (640x480)
- Create Buffer: FF D8 (SOI) + FF C0 (SOF0) marker. Segment length at offset 4 (UInt16BE, e.g. 0x0011).
  Height at offset 7 (UInt16BE: 480), width at offset 9 (UInt16BE: 640). Pad to 30+ bytes.
  Note: SOF0 structure = FF C0 LL LL PP HH HH WW WW ... so offset+5=precision, offset+5,6=height, offset+7,8=width.
  Full layout: [FF D8 FF C0 00 11 08 01 E0 02 80 ...pad...]
  height=0x01E0=480, width=0x0280=640
- Assert: `{ width: 640, height: 480 }`

Test 3: WebP lossy VP8 (320x240)
- Create Buffer: "RIFF" + 4 bytes file size + "WEBP" + "VP8 " + chunk.
  Width at byte 26 (UInt16LE: 320 & 0x3FFF), height at byte 28 (UInt16LE: 240 & 0x3FFF). Pad to 30+ bytes.
- Assert: `{ width: 320, height: 240 }`

Test 4: WebP lossless VP8L (150x75)
- Create Buffer: "RIFF" + 4 bytes + "WEBP" + "VP8L" + chunk.
  At byte 21: UInt32LE encoding (width-1) in bits 0-13, (height-1) in bits 14-27.
  For 150x75: bits = (150-1) | ((75-1) << 14) = 149 | (74 << 14) = 149 | 1212416 = 1212565
- Assert: `{ width: 150, height: 75 }`

Test 5: WebP extended VP8X (1920x1080)
- Create Buffer: "RIFF" + 4 bytes + "WEBP" + "VP8X" + chunk.
  Canvas at bytes 24-29: width-1 as 3 bytes LE, height-1 as 3 bytes LE.
  1919 = 0x077F -> bytes 24,25,26 = 0x7F, 0x07, 0x00
  1079 = 0x0437 -> bytes 27,28,29 = 0x37, 0x04, 0x00
- Assert: `{ width: 1920, height: 1080 }`

Test 6: GIF (80x60)
- Create Buffer: "GIF89a" + width 80 at byte 6 (UInt16LE) + height 60 at byte 8 (UInt16LE). Pad to 30+ bytes.
- Assert: `{ width: 80, height: 60 }`

Test 7: null/tiny buffer returns null
- Assert: `parseImageDimensions(null) === null`
- Assert: `parseImageDimensions(Buffer.alloc(5)) === null`

Test 8: Unrecognized format returns null
- Create buffer with random bytes (not matching any magic number).
- Assert: returns null

Helper function for tests: extract wrapClass from buildElementorJson result:
```javascript
function extractWrapClass(result) {
  const json = JSON.stringify(result);
  const match = json.match(/el-page-wrap-[0-9a-f]{6}/);
  return match ? match[0] : null;
}
```
